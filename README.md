## How to Install

1. [Click here to download the mod file directly](https://raw.githubusercontent.com/bo7med-x/cod1-interface-remake/main/main/zzz_bo7medx_ui_v2.pk3).
2. Go to your Call of Duty 1 installation folder.
3. Open the `main` directory.
4. Drop the `.pk3` file inside it.
5. Run the game.
